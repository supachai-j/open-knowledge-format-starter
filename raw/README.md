# raw/ — Source materials (Layer 1)

Drop original sources here: PDFs, meeting notes, exports, datasets, web captures.

- **Immutable.** The agent reads from `raw/` but never edits these files. This is your source of truth.
- Files here are NOT part of the OKF bundle (the bundle is `wiki/`). They are the inputs to ingestion.
- To bring a source into the knowledge base, trigger the **INGEST workflow** in `../AGENTS.md`
  (supervised — review the extracted claims before anything is written to `wiki/`).
